# dariusvlok30.github.io

Portfolio website for Darius Vlok — Full Stack AI Engineer.

Built with React, Tailwind CSS, and deployed via GitHub Pages.

## Development

```bash
yarn install
yarn start
```

## Deploy

Pushes to `main` automatically trigger a GitHub Actions build and deploy to GitHub Pages.

> **GitHub Pages setup:** In your repo Settings → Pages → Build and deployment, set the source to **GitHub Actions**.
